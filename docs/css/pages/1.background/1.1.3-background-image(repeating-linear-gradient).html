1.background-image:gradients

1.1 repeating-linear-gradient

1.1.1基础

repeating-linear-gradient是linear-gradient的循环版本。当色块不满足填充模块时，会自动循环填充剩余的内容。

语法

background-image:repeating-linear-gradient(<same linear-gradient>)

linear-gradient文档地址

示例

html 

<div class="box repeating-linear-gradient-default"></div>

css

.repeating-linear-gradient-default{
    background-image:repeating-linear-gradient(blue 0 10px,pink 10px 20px)
}
