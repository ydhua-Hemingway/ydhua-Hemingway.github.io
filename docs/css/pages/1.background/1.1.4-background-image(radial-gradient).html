1.background-image:gradients

1.1 radial-gradient

1.1.1基础

radial-gradient径向渐变。 先解释下是什么是径向，就是中心店延半径向外延伸方向。默认显示为圆。

语法

background-image: radial-gradient(<color>, <color>);

示例

html 

<div class="box radial-gradient-default"></div>

css

.radial-gradient-default{
    background-image:radial-gradient(blue,pink)
}

1.1.2 改变中心点位置

语法

background-image: radial-gradient(at <postion:x> <position:y>,<color>, <color>);

示例

html 

<div class="box radial-gradient-position-center"></div>

css

.radial-gradient-position-center{
    background-image:radial-gradient(at 0 10px,blue,pink)
}

1.1.3 改变径向渐变距离

语法

background-image: radial-gradient(<color> <position> <position>, <color> <position> <position>);

示例

html 

<div class="box radial-gradient-position-radial"></div>

css

.radial-gradient-position-radial{
    background-image:radial-gradient(yellow 10px,blue 10px 20%,pink)
}
