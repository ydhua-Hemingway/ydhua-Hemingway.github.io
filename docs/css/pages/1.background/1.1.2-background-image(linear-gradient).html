1.background-image:gradients

1.1 linear-gradient

1.1.1基础

linear-gradient为线性变化，其默认的方向为↓。

语法

background-image:linear-gradient(<color>,<color>...)

示例

html 

<div class="box linear-gradient-default"></div>

css

.linear-gradient-default{
    background:linear-gradient(blue,pink)
}

1.1.2 改变渐变方向

linear-gradient支持的方向改变属性
单一方向:to left、bottom(default)、top、right
多方向组合:to left bottom、left top...
角度改变:Angles。0角度位6点方向，角度顺时针增加。角度的详细属性参考Angles

语法

background:linear-gradient(<direction>,blue,pink)

示例

单一方向

组合方向

角度

html 

//单一方向
<div class="box linear-gradient-default"></div>
//组合方向
<div class="box linear-gradient-groupDirection"></div>
//角度
<div class="box linear-gradient-anglesDirection"></div>

css

/*单一方向 */
.linear-gradient-oneDirection{
    background:linear-gradient(to right,blue,pink)
}
/*组合方向 */
.linear-gradient-groupDirection{
    background:linear-gradient(to right top,blue,pink)
}
/*角度 */
.linear-gradient-anglesDirection{
    background:linear-gradient(30deg,blue,pink)
}

1.3设置颜色范围

从上文中我们发现不设置颜色的范围时，颜色是平均分配在视图中的，那么有没有办法去自定义颜色的范围呢，既然都这么问了，那么答案肯定是有的。

语法

background:linear-gradient(<color> <position> <position>,pink)

需要注意的是,第一个颜色起点永远为0，最后一个颜色终点永远为100%。 第一个position代表起点位置，第二个代表终点位置。也可以只设置一个position，那么就表明颜色的起点。 如果前一个颜色的终点与下一个颜色的起点一致，那么两个色块之间就不会产生过度渐变。也就是理解为两个色块之间的距离则为过度距离。

示例

html 

<div class="box linear-gradient-position"></div>

css

.linear-gradient-position{
    background:linear-gradient(blue 20% 30%,pink 30%,red 80%,yellow 90%)
}
